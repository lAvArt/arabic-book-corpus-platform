# Governance

## Intake Policy

- Curated maintainers only can ingest/publish.
- Community can propose books through issue templates.

## Publication Rules

No release is publishable unless all are true:

1. Source and edition provenance fields complete.
2. License and attribution metadata validated.
3. QA thresholds passed.
4. Open review tasks for release scope resolved.
5. Manifest checksum generated.

## Cadence

- Monthly immutable release drops.
- Each release has:
  - `version_tag`
  - changelog
  - checksum manifest
